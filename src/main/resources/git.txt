将项目通过git来进行管理
1.自己创建好基本项目结构
2.将我们的项目变成一个git仓库 -> git init
3.因为代码仓库只需要存放代码，想class文件、编译器自身生成的文件都不需要放入仓库中，而且
    如果放入 还会经常出现代码冲突。解决方案：使用忽略文件.gitignore 在文件中设置
4.在设置好忽略文件后 我们就可以将我们的代码添加到我们的本地仓库中
    这时对于我们的文件我们需要先通过 git add 要添加的资源路径 来对要放入本地仓库的代码
    进行添加操作，接下来通过git commit -m "提交信息" 来将项目代码提交到本地仓库，这时候才
    是正的将代码保存在本地仓库中 我们如果希望跟别人共同进行代码的管理就需要将我们本地仓库和远程
    github仓库进行连接 并将代码推送到远程服务器上 这样才能跟别人进行共同开发
    git remote add origin 自己的git仓库地址
    这样连接好之后我们就可以进行项目从本地到服务器的推送了
    git push -u origin master
